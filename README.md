# Task 3 – Interactive Dashboard Development  
## Apple Financial Performance Dashboard (2009–2024)

# Project Overview
This project was completed as part of the CODTECH Data Analytics Internship (Task 3 – Dashboard Development). 
The objective of this task was to design and develop a fully interactive dashboard to visualize Apple’s financial performance from 2009 to 2024.

The dashboard was built using Power BI and focuses on transforming raw financial data into meaningful and visual insights. 
The goal of this project is to enable users to quickly analyze trends, compare performance metrics and make data-driven interpretations through interactive visualizations.

This project demonstrates the practical application of data visualization principles and business intelligence tools in financial analysis.

## Objective of the Dashboard
The primary objectives of this dashboard are:
- To visualize Apple’s historical financial performance.
- To identify trends in revenue, profit and assets.
- To compare key financial metrics over time.
- To present financial insights in a clear and interactive manner.
- To support decision-making through visual analytics.
The dashboard provides a user-friendly interface where users can filter, explore and interpret financial data dynamically.

## Dataset Description
The dataset used in this dashboard includes Apple’s annual financial records from 2009 to 2024. The dataset contains the following key financial indicators:
- Revenue (millions)
- Net Income (millions)
- Gross Profit (millions)
- Operating Income (millions)
- Total Assets (millions)
- Cash on Hand (millions)
- Long Term Debt (millions)
- Number of Employees
- Earnings Per Share (EPS)
The data was cleaned and structured before importing into Power BI to ensure accurate visualization and analysis.

## Dashboard Features & Visualizations
The dashboard includes multiple interactive components designed to provide comprehensive financial insights:
# 1. KPI Cards
Key Performance Indicators (KPIs) are displayed at the top of the dashboard to provide a quick summary of:
- Total Revenue
- Total Net Income
- EPS
- Total Assets
These KPI cards allow users to instantly understand the overall financial position.

# 2. Revenue Trend Analysis
A line chart is used to display revenue trends over time. This visualization helps identify:
- Growth patterns
- Peak performance years
- Revenue fluctuations
Users can observe how revenue has evolved between 2009 and 2024.

# 3. Profitability Analysis
Bar charts and line graphs are used to compare:
- Gross Profit
- Operating Income
- Net Income
This section highlights Apple’s profitability trends and operational efficiency.

# 4. Assets & Debt Comparison
A comparative chart shows:
- Total Assets
- Long Term Debt
This helps analyze financial stability and capital structure over time.

# 5. Interactive Filters (Slicers)
The dashboard includes slicers that allow users to:
- Filter data by year
- Focus on specific financial metrics
- Dynamically update charts based on selections
This interactivity enhances user experience and analytical depth.

## Insights Derived from the Dashboard
The dashboard reveals several important insights:
- Apple shows consistent long-term revenue growth.
- Net income trends closely follow revenue growth patterns.
- EPS increases reflect improved shareholder value.
- Total assets show expansion, indicating business growth.
- Long-term debt trends provide insights into financial leverage strategy.
The visual format makes it easier to identify patterns that may not be immediately visible in raw data tables.
